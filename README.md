finance
=======

**Python for Financial Data Analysis with pandas**

Functions
---------

* JpStock - Scraping stocks data.
* OhlcPlot - Plotting OHLC Chart.
* TechnicalIndicators - Compute Technical Analysis.
* FileIO - Handling File I/O.
* Draw - Draw Charting.
* Analysis - Analysis codes.
